## Description

A NestJs backend API for Masadamsahid's Portfolio Website

## Installation

```bash
$ npm install
```

or

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
